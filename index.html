项目简介

French AI Learning Hub 是一个功能丰富的法语学习平台，集成了新闻阅读、影视推荐、词典查询、语用积累和AI助手等功能。

部署平台：Railway（使用 PostgreSQL 数据库）

数据同步：支持跨设备数据同步，用户数据存储在云端 PostgreSQL 数据库

使用的网络资源

📰 新闻模块

  • France 24: https://www.france24.com/fr/rss
  • Le Monde: https://www.lemonde.fr/rss/une.xml
  • RFI: https://www.rfi.fr/fr/rss
  • franceinfo: https://www.franceinfo.fr/rss
  • 20 Minutes: https://www.20minutes.fr/rss/une.xml

🎬 影视模块

  • TMDB API: https://www.themoviedb.org/
  • 用于获取法语电影和剧集信息、海报、评分等
  • 需要配置 TMDB_API_KEY 环境变量
  • 遵循 TMDB API 使用条款和 Attribution 要求

🤖 AI服务

  • DeepSeek API: https://platform.deepseek.com/
  • 用于新闻关键词提取和语用表达生成
  • 需要配置 DEEPSEEK_API_KEY 环境变量
  • Coze API: https://www.coze.cn/
  • 用于AI助手对话功能，提供流式响应
  • 需要配置 COZE_BOT_ID 和 COZE_PAT_TOKEN 环境变量

📚 词典数据

  • French-Dictionary: https://github.com/hbenbel/French-Dictionary
  • 基于Wiktionary的完整法语词典
  • 许可证: MIT License (详见项目 LICENSE-French-Dictionary 文件)
  • 版权: Copyright (c) 2021 Hussem Ben Belgacem

📄 开源许可

  • French-Dictionary: MIT License
  • 本项目使用的所有开源资源均遵循其原始许可证要求

技术栈

  • 前端: JavaScript (ES6+), Tailwind CSS
  • 后端: Flask (Python)
  • 数据库: Railway PostgreSQL
  • 数据存储: PostgreSQL（用户数据、历史记录、收藏等）+ 本地存储（localStorage/IndexedDB，用于缓存）
  • 部署: Railway
  • 认证: JWT Token + bcrypt 密码加密

功能特性

  • ✅ 用户注册和登录（使用注册码）
  • ✅ 跨设备数据同步（词典历史、收藏、学习进度等）
  • ✅ AI 聊天记录云端存储
  • ✅ 电影收藏列表云端存储
  • ✅ 语用表达和收藏云端存储
  • ✅ 背单词进度云端同步

数据说明

本项目使用以下数据源，所有数据均遵循其原始许可证要求：

  • 词典数据：基于 French-Dictionary (MIT License)，来源：Wiktionary
  • 电影数据：TMDB API，遵循 TMDB API 使用条款
  • 新闻数据：通过 RSS 订阅获取，仅用于学习目的
  • 用户数据：存储在 Railway PostgreSQL 数据库，遵循隐私保护原则
