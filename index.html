项目简介

French AI Learning Hub 是一个功能丰富的法语学习平台，集成了新闻阅读、影视推荐、词典查询、语用积累和AI助手等功能，旨在为法语学习者提供一站式的学习体验。

部署平台：Railway（使用 PostgreSQL 数据库，支持自动部署和扩展）

数据同步：支持跨设备数据同步，用户数据（词典历史、收藏、学习进度、AI对话记录等）存储在云端 PostgreSQL 数据库

词典数据：公共法语学习词典（从 FreeMDict 论坛 下载），包含完整的分词性词典文件（名词、动词、形容词、副词等），总计超过 11 万词条

AI 能力：集成 DeepSeek API（关键词提取、语用生成）和 Coze API（智能对话助手），提供流式响应体验

使用的网络资源

📰 新闻模块

  • France 24: https://www.france24.com/fr/rss
  • Le Monde: https://www.lemonde.fr/rss/une.xml
  • RFI: https://www.rfi.fr/fr/rss
  • franceinfo: https://www.franceinfo.fr/rss
  • 20 Minutes: https://www.20minutes.fr/rss/une.xml

🎬 影视模块

  • TMDB API: https://www.themoviedb.org/
  • 用于获取法语电影和剧集信息、海报、评分等
  • 需要配置 TMDB_API_KEY 环境变量
  • 遵循 TMDB API 使用条款和 Attribution 要求

🤖 AI服务

  • DeepSeek API: https://platform.deepseek.com/
  • 用于新闻关键词提取和语用表达生成
  • 需要配置 DEEPSEEK_API_KEY 环境变量
  • Coze API: https://www.coze.cn/
  • 用于AI助手对话功能，提供流式响应
  • 需要配置 COZE_BOT_ID 和 COZE_PAT_TOKEN 环境变量

📚 词典数据

  • 数据来源：公共法语学习词典（从 FreeMDict 论坛 下载）
  • 数据规模：超过 11 万词条，分词性存储（名词、动词、形容词、副词、连词、介词、代词、限定词）
  • 文件格式：JSON 格式，存储在 public/data/dicts/ 目录
  • 内容：包含词性、性别、变位、定义、例句等完整信息

📄 开源许可

  • French-Dictionary: MIT License
  • 本项目使用的所有开源资源均遵循其原始许可证要求

技术栈

前端技术

  • JavaScript (ES6+)：原生 JavaScript，模块化设计
  • Tailwind CSS：实用优先的 CSS 框架，响应式设计
  • LocalStorage/IndexedDB：本地数据缓存，提升性能

后端技术

  • Flask (Python 3.11+)：轻量级 Web 框架
  • Gunicorn：生产环境 WSGI 服务器
  • PostgreSQL：关系型数据库，存储用户数据
  • JWT Token：用户认证和授权
  • bcrypt：密码加密存储

AI 服务

  • DeepSeek API：新闻关键词提取、语用表达生成
  • Coze API：智能对话助手，流式响应

部署与运维

  • Railway：云平台部署，自动构建和扩展
  • Docker：容器化部署（可选）
  • Nixpacks：自动构建系统

功能特性

📖 词典模块

  • 快速搜索法语词汇（支持完整 Wiktionary 词典，超过 11 万词条）
  • 智能索引系统：精确匹配、前缀匹配、模糊匹配
  • 自动保存查询历史，支持跨设备同步
  • 收藏夹功能，收藏常用单词
  • 背单词系统：随机学习、复习生疏单词、熟练度标记（生疏/模糊/熟练）
  • 已学列表管理，支持快速切换熟练度
  • 分词性词典：名词、动词、形容词、副词、连词、介词、代词、限定词

📰 新闻模块

  • 自动抓取法国主流媒体最新新闻（France 24、Le Monde、RFI等）
  • 支持关键词提取和翻译（DeepSeek API）
  • 新闻列表实时更新

🎬 影视模块

  • 推荐高质量法语电影和剧集（TMDB API）
  • 显示评分、简介、海报等信息
  • 支持"想看"列表管理，云端同步
  • 自动翻译剧情简介

💬 语用积累

  • 学习常用法语表达和短语
  • AI生成情景对话（DeepSeek API）
  • 收藏喜欢的表达，云端同步

🤖 AI助手

  • 基于Coze平台的智能对话
  • 支持法语学习相关问题
  • 流式响应，体验流畅
  • 聊天记录云端存储，跨设备同步

👤 用户系统

  • 用户注册和登录（使用注册码）
  • JWT Token 认证，安全可靠
  • 跨设备数据同步（词典历史、收藏、学习进度、AI对话、电影收藏、语用收藏等）
  • 所有用户数据存储在云端 PostgreSQL 数据库

数据说明

本项目使用以下数据源，所有数据均遵循其原始许可证要求：

📚 词典数据

  • 来源：公共法语学习词典（从 FreeMDict 论坛 下载）
  • 内容：分词性词典文件（adj.json, adv.json, conj.json, det.json, noun.json, prep.json, pron.json, verb.json）
  • 规模：超过 11 万词条，包含词性、性别、变位、定义、例句等信息

🎬 电影数据

  • 来源：TMDB API (The Movie Database)
  • 内容：法语电影和剧集信息、海报、评分、简介等
  • 使用条款：遵循 TMDB API 使用条款和 Attribution 要求
  • 配置：需要配置 TMDB_API_KEY 环境变量

📰 新闻数据

  • 来源：通过 RSS 订阅获取法国主流媒体新闻
  • 媒体：France 24、Le Monde、RFI、franceinfo、20 Minutes 等
  • 用途：仅用于学习目的，不用于商业用途

👤 用户数据

  • 存储：Railway PostgreSQL 数据库
  • 内容：用户账号、词典历史、收藏、学习进度、AI对话记录、电影收藏、语用收藏等
  • 安全：密码使用 bcrypt 加密，遵循隐私保护原则
  • 同步：支持跨设备数据同步
