项目简介

French AI Learning Hub 是一个功能丰富的法语学习平台，集成了新闻阅读、影视推荐、词典查询、语用积累和AI助手等功能。

使用的网络资源

📰 新闻模块

  • France 24: https://www.france24.com/fr/rss
  • Le Monde: https://www.lemonde.fr/rss/une.xml
  • RFI: https://www.rfi.fr/fr/rss
  • franceinfo: https://www.franceinfo.fr/rss
  • 20 Minutes: https://www.20minutes.fr/rss/une.xml

🎬 影视模块

  • TMDB API: https://www.themoviedb.org/
  • 用于获取法语电影和剧集信息、海报、评分等

🤖 AI服务

  • DeepSeek API: https://platform.deepseek.com/
  • 用于新闻关键词提取和语用表达生成
  • Coze API: https://www.coze.cn/
  • 用于AI助手对话功能，提供流式响应

📚 词典数据

  • French-Dictionary: https://github.com/hbenbel/French-Dictionary
  • 基于Wiktionary的完整法语词典

技术栈

  • 前端: JavaScript (ES6+), Tailwind CSS
  • 后端: Vercel Serverless Functions (Python)
  • 数据库: Supabase PostgreSQL
  • 认证: JWT Token
  • 部署: Vercel + Supabase
